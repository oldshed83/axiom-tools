# NYRUS
### The Axiom Upgrade
Simple, fast, and custom-tuned for Axiom.
##
Features
- **Pro Filters:** Catch the plays you want without the noise.
- **Instant Gas:** One-click priority fees for fast execution.
- **Zero Install:** Syncs directly to your terminal via bookmark.
- **Secure:** Encrypted settings sync.
##
Getting Started
1. Open the Nyrus dashboard.
2. Drag the **Sync Button** to your bookmarks bar.
3. Go to **Axiom.trade** and click the bookmark.
Done. Your terminal is now upgraded.
